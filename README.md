# IronOCR-Goruntu-Metin-Ceviri
Görüntü üzerindeki yazılı dokümanları yazı(text) haline dönüştürme

## Dependancies

Programın çalışması için "IronOcr" eklentisini kurmanız gerekmektedir.

```
 PM > Install-Package IronOcr
```
